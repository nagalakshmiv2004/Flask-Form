# Flask Form

## Overview
Simple form to complete the google code-in task
https://codein.withgoogle.com/dashboard/task-instances/5691385028018176/

## Pre-Requisite

    1. python 3.x
    2. pip
    3.Flask


## Installing Flask

    1. pip install virtualenv
    2. venv\Scripts\activate
    3. pip install Flask

## How To Run

set FLASK_APP=Form.py
Flask run
